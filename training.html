Coding Languages

In my training to become a professional Front-End Web Developer in the Treehouse tech degree program, I learned the following coding languages in addition to becoming proficient using the following coding programs. I will use all this training, knowledge, and experience to build your website.

HTML 5

HTML is the base of everything on the web. It is the programing language developers use to create the content layer on any website.

CSS 3

CSS or Cascading Style Sheets makes the web beautiful. It is the programming language developers use to style the content created in HTML.

Sass

Sass is a programing language developers use to create smarter, more condensed CSS. Sass allows developers to style websites with faster load times.

JavaScript

JavaScript makes the web the engaging place we all love. It is the programing language developers use to make both HTML and CSS more interactive.

jQuery

jQuery is a programing language developers use to write fast and smarter Javascript. Just as Sass increases the load speed of a page, jQuery does the same.

Programing Software

Atom

Front-End Developers, as well as other programmers, write their code in a code editor. Atom is a code editor that many professional developers use.

Slack

Slack is a messaging program many professional web developers use. It allows Web Designer, Front-End Developers, and Back-End Developers to collaborate.

PrePros

PrePros is a tool many developers use to assist them in turning Sass into condensed CSS.

Github

Github allows professional developers to upload the code they are working on to a central server enabling other members of their team to access their code.

Bootstrap

Bootstrap allows professional developers to the ability to write faster HTML. With its pre-programmed settings, developers can also quickly style their content.
